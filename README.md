# Crawler Signal Reference Library

A comprehensive 30-document reference for every signal a web crawler reads from a site. Authored by [Joseph W. Anady](https://www.thatdevpro.com), founder of [ThatDevPro](https://www.thatdevpro.com) (engine optimization studio) and [ThatDeveloperGuy](https://thatdeveloperguy.com) (SDVOSB-certified veteran-owned web + AI engineering studio).

## Coverage

### HTML signals (18 documents)
- App + PWA metadata
- Canonical + alternate
- Favicon + icons
- Meta: author, charset, color-scheme, content-language, copyright, generator, keywords, referrer, refresh, robots, theme-color, viewport
- Open Graph protocol
- Twitter Cards
- Search engine verification (Google, Bing, Yandex, IndexNow, etc.)

### HTTP signals (12 documents)
- Status codes: 2xx, 3xx, 4xx, 5xx
- Headers: caching, content, CORS, performance, rate-control, request, security, SEO
